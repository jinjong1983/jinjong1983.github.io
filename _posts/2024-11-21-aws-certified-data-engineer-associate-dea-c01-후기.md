# DEA
올해였든가 나온지 얼마 안된 시험이다. 기존에 aws 자격중에는 data analytics 스페셜리티가 있었는데 해당 자격이 사라지고 dea "asso" 가 생겼다. mla asso 와 ai practioner 가 생겼기 때문에 dva,soa 위에 dop 상위 자격이 있듯이 mla asso 와 dea asso 위에 아마 ㅇpro 자격증이 생기지 않을까 예상한다. 


# 효용성
시험자체는 aws 기반의 파이프라인 구축, 데이터 분석 에 대한 내용이 주로 나오는 시험으로 공부하는 과정에서 재미도 있었고 본업과 유사하여 좀 수월하게 공부할 수 있었다. glue, athena, redshift 에 대한 내용이 많았고 dynamodb stepfunction watch lambda kinesis시리즈 에 대한 내용도 나온다. quicksight lakeformation등에 대한 내용도 나오고 기본적인 s3 iam 등에 대한 내용도 나온다. 핵심은 glue athena s3 

# 공부방법
마렉.. 그리고 나온지 얼마 안되어 연습문제나 덤프를 보면서 감을 잡으면 어렵지 않다. 특히 데이터쪽 종사자라면 soa 보단 좀 수월하다. 개인적으론 dva,dea 가 좀 쉽고 saa 는 범위의 압박이 있으며 soa 는 아예 내가 아는거 aws가 맞나 싶은 정도였다.

이거로 clf 이후 네번째 자격증이고 다음주에 aif 까지 보면 총 다섯개.
